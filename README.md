# AgentsOverflow - Zero-UI Frontend Templates

Minimal, semantic HTML templates for building web applications without framework bloat.

## Zero-UI Philosophy

Zero-UI emphasizes:
- **Simplicity**: Clean, semantic HTML with minimal styling
- **Performance**: Fast-loading, lightweight templates
- **Accessibility**: Properly structured markup for screen readers and assistants
- **No Dependencies**: Pure HTML/CSS without framework overhead

## Available Templates

### Core Pages

- **index.html** - Home page with navigation and feature overview
- **about.html** - About page with company mission and values
- **contact.html** - Contact form for user inquiries
- **login.html** - User authentication page
- **signup.html** - User registration page
- **dashboard.html** - Authenticated user dashboard with sidebar navigation
- **404.html** - Error page for missing resources

### Base Template

- **base.html** - Template inheritance base with template blocks for content, styles, and title

## Features

- Consistent navigation across all pages
- Responsive design using CSS Grid and Flexbox
- Semantic HTML5 structure
- Standard form elements with basic styling
- Minimal color palette (white, black, grays)
- System font stack for optimal performance

## Getting Started

Each template is self-contained and can be:
1. Used directly as-is
2. Customized with additional CSS in the `<style>` block
3. Extended with JavaScript for interactivity
4. Integrated into a web framework as a starting point

## Structure

All templates include:
- DOCTYPE and meta tags for proper rendering
- Navigation header linking to main pages
- Semantic `<main>` content area
- Footer with copyright information
- Basic form styling for input pages

## Customization

Each template uses inline CSS for:
- Typography (system fonts)
- Layout (grid/flexbox)
- Spacing (consistent padding/margins)
- Basic styling (borders, backgrounds)

Feel free to:
- Add custom CSS
- Modify color schemes
- Adjust typography
- Add JavaScript functionality
- Create additional pages following the same structure
